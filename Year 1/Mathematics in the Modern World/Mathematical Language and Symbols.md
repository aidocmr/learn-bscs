# Language of Sets

## What are Sets?
A set is a collection of well defined objects. Usually denoted by capital letters of the alphabet and its members are enclosed with curly braces.
### Elements Inside a Set
Elements are the members or objects of the set which is denoted by a symbol (∈) *($\LaTeX{}$: `\in`)*
## Set Theory
This is a branch of mathematics that studies sets or the mathematical science of infinite.
## Two ways of Representing a Set
### Raster Method (Tabulation)
Elements are enumerated and separated by a comma.
$$U = \{a,s,d,f,g,h,j,k,l\}$$
### Rule Method (Set Builder Notation)
Used to describe the elments or members of the set using their common characteristics.
$$U = \{x|x are letters in my keyboard\}$$
### Cardinal Number
Refers to the number of elementsi n a given set. The cardinality of a set is given by n(A)
$$U = \{a,s,d,f,g,h,j,k,l\}$$
$$U = 9$$
## Types of Set
### Finite Set
Is a set whose elements are limited or countable and the last element can be identified.
### Infinite Set
Is a set whose elements are unlimited or uncountable and the last element cannot be specified.
### Unit Set
Is a set with only one element
### Empty Set
Unique set with no elements and also called as the **Null Set*. It is denoted by $\{\}$
### Universal Set
The totality of the set, all sets under investigation in any application of set theory are assumed to be contained in some largely fixed set and is denoted by $U$
### Subset
If $A$ and $B$ are set, $A$ is called a subset of $B$, written $A \subseteq B$, if and only if every element of $A$ is also an element of $B$. A proper subset of $B$, written $A \subset B$, if and only if, every element of $A$ is in $B$ but there is at least one element of $B$ that's not in $A$.
### Equal Set
Two sets are equal if and only if, every element of $A$ is in $B$ and every element of $B$ is in $A$.
### Equivalent Set
Two sets are equivalent if they have the same number of elements and it is denoted by (~)
### Disjoin Set
Two sets that do not have the same elements. This is also known as a non-intersecting set.
## Operations on Sets
### Union of Set
The union of $A$ and $B$, denoted by $A \cup B$ ($\LaTeX$: `\cup`)
### Intersection of Set
The intersection of $A$ and $B$, denoted by $A \cap B$ ($\LaTeX$: `\cap`)
### Complement of Set
The complement of a set or absolute complement $A$, denoted by $A'$, is the set of all elements in $x$ in $U$ such that $x$ is not in $A$ (Not an element of $A$)
### Difference of Set
The difference of $A$ and $B$, denoted by $A-B$, is the set of all possible ordered pairs with first element from $A$ and second element from $B$
